# xBet
xBet是一个面向体育竞猜、预测市场、资金钱包、区块链投注、代理分佣、内容资讯与用户互动的一体化投注平台。
## 1. 项目架构

系统由以下应用组成：

**Api**：统一业务 API 服务

**UserWeb**：用户端

**AdminWeb**：管理端

**App**：Android & iOS App 应用端

## 2. 系统功能模块说明

### 2.1 用户账号体系（`usr`）

普通用户/代理账号、KYC验证信息、角色权限体系等；用户注册、登录、身份验证、权限控制、实名认证、登录安全控制等功能。支持KYC 审核信息、黑名单与风控拦截。

### 2.2 用户钱包体系（`fin`）

钱包账户、交易记录、充值/提现申请、链上加密货币、汇率管理、转账记录；覆盖余额、冻结、充值、提现、投注扣款、结算返奖等全过程。

### 2.3 赛事子系统（`sport`）

足球赛事(洲际/国家/联赛/球队/比赛/赛事事件)；单一事件投注(PolyMarket风格)；

支持赛事: 世界杯/世俱杯/欧洲杯/美洲杯/非洲杯/亚洲杯欧冠/主流五大联赛等。

### 2.4 投注与结算子系统（`bet`）

包含: 投注市场类型、比赛赔率、投注单、投注项目、结算记录。

支持投注类型: 胜平负/亚盘/大小球/比分/角球/红黄牌/半场/滚球等。

### 2.5 代理与分佣系统（`agent`）

包含: 代理等级定义、流水返点阶梯、等级阶梯、返点规则、代理档案、佣金记录、代理层级关系(上下级代理)、月度统计

设计参考: 主流博彩平台代理返点(Rebate)机制

返点模型: 流水返点(Volume Rebate) + 等级阶梯(Level Tier) 双引擎组合

### 2.6 统计分析（`stats`）

包含: 平台日报统计、用户投注统计、赛事投注统计、代理日统计、财务汇总维度。

### 2.7 日志与审计（`log`）

包含: 用户操作日志、系统错误日志、API请求日志、管理员审计日志、关键数据变更日志。

### 2.8 系统配置（`cfg`）

包含: 系统配置参数、邮件模板、通知队列、公告管理、横幅管理、外部API配置

### 2.9 内容媒体子系统（`media`）

包含:[新闻资讯模块][比赛直播模块]

### 2.10 聊天与互动子系统（`chat`）

在线聊天子系统

-- [会话模块]聊天会话(用户↔代理 / 用户↔客服)
-- [消息模块]聊天消息(文字/表情/附件)
-- [附件模块]消息附件(图片/文件)
-- [已读模块]消息已读状态
-- [快捷模块] 客服/代理快捷回复模板
